# Following-the-Money-U.S.-Election-Finance-Insights

A Tableau Data Story on How Money Shapes Presidential Elections

This project explores how billions of dollars in political donations influence U.S. presidential elections. Using campaign finance data, the analysis uncovers donation patterns, geographic funding trends, candidate war chests, campaign spending behaviors, and whether financial power truly determines election outcomes.

All dashboards in this project were designed and built using Tableau.
